📚 使用教程与设置指南

🔑 1. 配置 API 和 URL

要使用 AI 总结功能，您需要配置 AI 服务商的 API Key。

  1. 在程序根目录下找到 src 文件夹。
  2. 打开 src/config.json 文件（推荐使用 VS Code 或记事本）。
  3. 找到 "ai" 部分，修改以下字段：
"ai": { "base_url": "您的 API 地址 (例如 https://api.openai.com/v1)", "api_key": "您的 sk-开头的密钥", "model": "gpt-4o-mini" }

🍪 2. 获取并配置 Cookie

为了让程序能够访问您的学校邮箱，需要提供登录后的 Cookie。

  1. 在 Chrome 或 Edge 浏览器中安装 Cookie-Editor 插件。
  2. 登录您的学校邮箱 (mail.xjtlu.edu.cn)。
  3. 点击浏览器右上角的 Cookie-Editor 图标。
  4. 点击右下角的 Export 按钮，选择 Export as JSON。
  5. 打开 src/config.json，把导出的 JSON 粘贴到 "cookies" 字段中。

注意：Cookie 会定期失效，如果程序无法登录，请重复此步骤更新。

📂 3. 文件结构说明

  • run_app.bat: 唯一的启动入口，双击即可运行。
  • src/config.json: 配置文件，在此修改 API 和 Cookie。
  • src/app.py: 核心服务程序。
