使用指南

1. 配置 AI 服务

点击顶部的“配置”，填写 AI Base URL 和模型名称。API Key 为可选（留空将自动使用系统托管的 AI 服务）。保存后立即生效。

"ai": { "base_url": "https://api.openai.com/v1", "api_key": "sk-... (可选)", "model": "gpt-4o-mini" }

2. 连接邮箱（Cookie）

推荐：在首页「连接邮箱」区域点击「打开邮箱登录窗口」，在本机 Edge 中完成登录后点「我已登录」。

高级：在 Chrome 或 Edge 安装 Cookie-Editor，登录邮箱后 Export as JSON，粘贴到「高级：直接粘贴 Cookie」文本框并保存。

如果 Cookie 失效，可重新走上述流程覆盖保存。

3. 使用方式

  • 关键词留空时，程序直接读取最近邮件。
  • 输入总结指令后，系统会搜索、提取正文并生成总结。
  • 查询结果和搜索历史会保存在本地浏览器存储中。
