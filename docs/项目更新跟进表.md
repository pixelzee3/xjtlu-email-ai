# 项目更新跟进表

用于对照 **Git 仓库当前提交（HEAD）** 记录工作区改动规模，便于和组员同步或写周报时快速引用。

## 使用说明

- **基准**：下列统计以本地 `HEAD` 为准（当前分支最后一次提交）。刷新方式：

```bash
git status --short
git diff --stat HEAD
git ls-files --others --exclude-standard
```

- 将本表中的「快照日期」在每次大改后手动更新，并粘贴最新 `git diff --stat` 输出到「附录」。
